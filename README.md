# Audi Pet Project

Учебный проект веб-сайта Audi с современным дизайном и интерактивными элементами.

## 📁 Структура проекта

```
audi.com/
├── index.html              # Главная страница
├── css/
│   ├── reset.css          # CSS reset для кроссбраузерности
│   └── style.css          # Основные стили проекта
├── fonts/
│   ├── auditype_extendedbold.ttf
│   └── auditype_extendednormal.ttf
└── images/
    ├── copy/              # Копии изображений
    ├── favicon/           # Иконки браузера
    └── original/          # Оригинальные изображения и видео
        ├── promo-photo*.png
        ├── topic-photo-*.png
        ├── audi-logo-header.png
        └── promo-video7.mp4
```

## 🎨 Возможности

- Адаптивный дизайн
- Использование фирменного шрифта Audi (AudiType)
- HD изображения и видео контент
- Современная структура HTML5
- CSS стили с кроссбраузерной совместимостью

## 🚀 Как запустить

1. Откройте `index.html` в браузере или используйте локальный сервер:
   ```bash
   # Если установлен Python 3
   python -m http.server 8000
   
   # Или используйте Live Server в VS Code
   ```

2. Откройте `http://localhost:8000` в браузере

## 📄 Файлы

- **index.html** - Основная страница сайта с разметкой контента
- **css/reset.css** - Нормализация стилей для всех браузеров
- **css/style.css** - Основные CSS правила, оформление и макет
- **fonts/** - Фирменные шрифты Audi для типографики
- **images/** - Медиа-ресурсы проекта (изображения, видео, иконки)

## 🔧 Технологии

- HTML5
- CSS3
- Растровая графика (PNG, JPG)
- Видео (MP4)
- Шрифты (TTF)
