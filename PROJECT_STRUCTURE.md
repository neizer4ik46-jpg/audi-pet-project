## Описание ключевых файлов

### index.html
Основной HTML файл проекта, содержит:
- Семантическую разметку HTML5
- Ссылки на CSS и шрифты
- Структуру боковой панели навигации
- Контент страницы с изображениями и информацией
- Мета-теги для оптимизации

**Ключевые элементы:**
- `<header>` - шапка с логотипом и навигацией
- `<nav>` - основное меню навигации
- `<main>` - основной контент
- Изображения и видео контент

### css/reset.css
Файл для нормализации стилей браузера:
- Удаляет стандартные отступы и межстрочные интервалы
- Устанавливает единообразные стили для всех браузеров
- Нормализует форматирование текста

### css/style.css
Основной файл стилей проекта:
- Определение цветовой схемы
- Адаптивный дизайн (media queries)
- Стили для типографики
- Оформление компонентов (кнопки, меню, блоки)
- CSS переменные для единообразия

### fonts/
Фирменные шрифты Audi:
- `auditype_extendedbold.ttf` - Жирный вариант для заголовков
- `auditype_extendednormal.ttf` - Обычный вариант для основного текста

**Использование в CSS:**
```css
@font-face {
    font-family: 'AudiTypeNormal';
    src: url('../fonts/auditype_extendednormal.ttf') format('truetype');
}

@font-face {
    font-family: 'AudiTypeBold';
    src: url('../fonts/auditype_extendedbold.ttf') format('truetype');
}
```

### images/
Организация изображений и медиа:
- **original/** - Высокое качество (HD) для хранения и экспорта
- **copy/** - Рабочие версии для использования в разработке
- **favicon/** - Иконки браузера разных размеров
